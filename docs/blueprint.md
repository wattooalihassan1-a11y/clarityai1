# **App Name**: Clarity AI

## Core Features:

- AI Chat with Context: Engage in natural language conversations, leveraging AI (Gemini) to maintain context, personalize the experience, and respond to text and image inputs, all while utilizing slash commands to trigger specialized capabilities such as study, explain, summarize, idea generation, and image generation; the LLM will use tools to decide which capabilities to employ
- Homework Helper: Provides step-by-step solutions and final answers to homework questions.
- Topic Explanation: Generates a clear explanation of a topic, along with examples and analogies.
- Text Summarization: Creates concise summaries of long pieces of text.
- Idea Brainstorming: Generates a list of creative ideas related to a specific topic.
- Voice Input & Image Upload: Enable voice-to-text using SpeechRecognition API and allow image uploads, displaying image previews above the input area.
- Chat History Management: Utilize localStorage to store, retrieve, and manage chat histories, including new chats, loading existing chats, deleting chats, and updating messages.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB), inspired by the concept of clarity and knowledge, evokes feelings of trust, calmness, and intelligence.
- Background color: Very light blue (#F0F8FF), same hue as the primary but desaturated to create a calm backdrop.
- Accent color: Lavender (#D0BFFF) a similar hue to primary for gentle contrast
- Headings: 'Sora' (sans-serif) - specified by user for a clean, modern look.
- Body: 'Manrope' (sans-serif) - specified by user, complements 'Sora' for readability.
- Lucide-react icons used throughout the application for a consistent and modern aesthetic.
- AppShell component provides a consistent header layout, including an app icon (BrainCircuit), app title ("Clarity AI"), and a byline.
